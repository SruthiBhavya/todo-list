To-Do List App (React)

A simple and responsive To-Do application built using React.
Users can add, edit, delete, and mark tasks as complete.
Tasks are stored in localStorage, so they remain even after refreshing the page.

Features:

1. Add new tasks
2. Edit existing tasks
3. Delete tasks
4. Mark tasks as Complete / Undo
5. Filter tasks (All / Active / Completed)
6. Data saved in localStorage
7. Clean and responsive UI
